Thông báo tuyển sinh lớp Java Web Fullstack

25 tháng 08, 2020

Tháng 9/2020, Techmaster Vietnam dự kiến khai giảng lớp lộ trình Java Web Fullstack, ngoài việc độc lập nền tảng,
phong cách "lập trình hướng đối tượng" của Java và sự hấp dẫn rất cao đối với các nhà tuyển dụng IT khiến Java trở
thành một ngôi sao sáng chói trên thị trường tuyển dụng việc làm hiện nay.

Nhu cầu tuyển dụng Java rất lớn thu hút số lượng lớn học viên đăng ký học Java

Đối tượng phù hợp với khóa học:

  • Sinh viên CNTT năm 2, năm 3, các bạn lập trình viên muốn học thêm về lập trình backend.
  • Đã có kiến thức căn bản về lập trình Web Front-end (HTML, CSS, Javascript), Git, SQL, Linux.
  • Đam mê tìm tòi, xử lý các logic nghiệp vụ cho doanh nghiệp.
  • Mong muốn tham gia xây dựng những hệ thống lớn, phục vụ hàng triệu người dùng.
  • Tư duy logic, toán, hệ thống CNTT tốt

Mục tiêu khóa học hướng tới:

  • Nắm vững kiến thức căn bản về Java. Đủ khả năng thi chứng chỉ Java Oracle 11
  • Sử dụng Spring Boot để dựng một ứng dụng web hoàn chỉnh theo mô hình MVC
  • Đóng gói và triển khai ứng dụng web lên Google Cloud bằng Docker

Lộ trình học cụ thể:

  • Java OOP (15 buổi)
  • Java nâng cao (15 buổi)
  • Docker Devops (6 buổi)
  • Java Spring Boot (18 buổi)

Khóa học kéo dài trong 4,5 tháng, một tuần học 3 buổi, mỗi buổi 3 giờ đồng hồ. Giáo trình Java chuẩn của đại học Hoa Kỳ + 1000 ví dụ mẫu.

Thời gian và địa điểm:

  • Thời gian học: 18:30 - 21:30 PM | thứ 3, thứ 5 và thứ 7
  • Ngày khai giảng: 8/9/2020
  • Địa điểm: tầng 12A, tòa nhà Viwaseen, số 48 Tố Hữu, Trung Văn, Nam Từ Liêm, Hà Nội.

Học phí: 14,000,000 VND / 4,5 tháng

Đặc biệt, khi đăng ký lớp, học viên sẽ được:

  • Thực hành phòng lab, giảng viên hướng dẫn từng bước từ dễ đến khóa
  • Hệ thống bài giảng trực tuyến gồm video, câu hỏi trắc nghiệm, bài tập hỗ trợ học viên ôn tập kiến thức.
  • Hỏi đáp, thảo luận trực tuyến qua Google Meet, Zalo, Skype.
  • Kho sách tham khảo cùng hàng trăm sản phẩm, đồ án mẫu từ các học viên cũ.
  • Tham gia các hoạt động dã ngoại (3 tháng/lần) cùng Techmaster

Cuối khóa, mỗi học viên sẽ phải bảo vệ đồ án cá nhân.
Nếu pass và có sản phẩm chất lượng sẽ được trung tâm hỗ trợ giới thiệu việc làm.
